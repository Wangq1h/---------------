## 数据预处理
数据预处理是这次设计处理的核心。在本模型中，分析讨论了以下几种预处理方法：
* 尺度统一化：`transforms.Resize(scale)`。
* 随机旋转：`transforms.RandomRotation(30)`。
* 随机亮度变化：`transforms.ColorJitter(brightness=0.2, contrast=0.2)`。
